# 🤝 Contributing to GhostRoute Pro

First off, **thank you** for considering contributing! 👻

## 📋 Code of Conduct
This project follows a [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold it.

## 🚀 How Can I Contribute?

### 🐛 Reporting Bugs
1. Check if the bug is already reported in [Issues](https://github.com/debjit604/ghostroute/issues)
2. If not, create a new issue using the **Bug Report** template
3. Include as much detail as possible

### 💡 Suggesting Features
1. Check if the feature is already requested in [Issues](https://github.com/debjit604/ghostroute/issues)
2. If not, create a new issue using the **Feature Request** template
3. Explain the use case clearly

### 🔧 Pull Requests
1. Fork the repository
2. Create a new branch: `git checkout -b feature/amazing-feature`
3. Make your changes
4. Test thoroughly
5. Commit: `git commit -m '✨ Add amazing feature'`
6. Push: `git push origin feature/amazing-feature`
7. Open a Pull Request
